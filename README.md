# 🌌Ultimate-mass-calculator
This is a simple project of mine which calculates how mass differ in different planets.  
It takes a mass in kg and convert it to the weight it would be in any planet in the solor system.

### ✒️Languages used
- HTML
- CSS
- JAVASCRIPT

### ❓Logics
- Taking the input and selecting it using queryselector.
- Calculating the weight using formula.
- Create elements by createElement function and store the values
- Displaying the weight and image of the planet using append child.
